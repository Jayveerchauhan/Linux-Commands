ls - The most frequently used command in Linux to list directories
>> ls
pwd - Print working directory command in Linux
>> pwd
cd - Linux command to navigate through directories
>> cd (directorry name) to go directory
>> cd .. (to go back one directory)
>> cd - (to go back last directory)
mkdir - Command used to create directories in Linux
>> mkdir (directory name)
mv - Move or rename files in Linux
>> mv (source name) (destination name)
>> mv file1.txt folder1
cp - Similar usage as mv but for copying files in Linux
>> cp (file name)
rm - Delete files or directories
>> rm(file name) to delete files
>> rm -rf (directory name) to delete directory 
touch - Create blank/empty files
>> touch file.txt
ln - Create symbolic links (shortcuts) to other files
cat - Display file contents on the terminal
>> cat file.txt 
clear - Clear the terminal display
echo - Print any text that follows the command
man - Access manual pages for all Linux commands
>> man ls
>> man cat
>> man cd 
uname - Linux command to get basic information about the OS
whoami - Get the active username
tar - Command to extract and compress files in Linux
>> tar -cvf (file_name) to compress
>> tar -xvf (file_name) to extract
grep - Search for a string within an output
>> cat (file_name) | grep (folder name)
>> cat file.txt | grep foder1
head - Return the specified number of lines from the top
>> head filename
tail - Return the specified number of lines from the bottom
>> tail filename
export - Export environment variables in Linux
zip - Zip files in Linux
unzip - Unzip files in Linux
ssh - Secure Shell command in Linux
service - Linux command to start and stop services
ps - Display active processes
kill and killall - Kill active processes by process ID or name
df - Display disk filesystem information
chmod - Command to change file permissions
ifconfig - Display network interfaces and IP addresses
wget - Direct download files from the internet
ufw - Firewall command
apt, pacman, yum, rpm - Package managers depending on the distro
sudo - Command to escalate privileges in Linux
alias - Create custom shortcuts for your regularly used commands
whereis - Locate the binary, source, and manual pages for a command
whatis - Find what a command is used for
top - View active processes live with their system usage
